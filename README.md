# Jade Stylus CoffeeScript

A Grunt project for a template of Jade, Stylus and CoffeeScript.

## Setup

Install Node.js v0.10.x at first.

```
npm install -g grunt-cli@0.1.9
npm install
```

## Building

```
grunt
```

`index.html`, `design.css` and `main.js` are generated.
